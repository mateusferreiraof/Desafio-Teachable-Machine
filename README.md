# Teachable Machine - Reconhecimento de Maçã

Aplicação web simples utilizando o Google Teachable Machine para reconhecer imagens de maçã.

## Descrição

O modelo foi treinado para identificar duas classes:

- Maçã
- Não é maçã

A aplicação utiliza o modelo treinado para tomar decisões no fluxo, exibindo a predição com a porcentagem de confiança.

## Tecnologias

- HTML
- CSS
- JavaScript
- TensorFlow.js
- Teachable Machine

## Como executar

1. Clone o repositório.
2. Abra a pasta no VS Code.
3. Abra o arquivo `index.html` com a extensão **Live Server**.
4. Selecione uma imagem de maçã para testar o modelo.
